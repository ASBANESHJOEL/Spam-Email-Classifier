# Spam Email Classifier

## Objective

Detect and classify messages as spam or non-spam using machine learning techniques and text processing.

## Dataset

SMS Spam Collection Dataset

## Technologies Used

- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer

## Algorithm Used

- Multinomial Naive Bayes

## Accuracy

96.23%

## Conclusion

A Spam Email Classifier was developed using the Multinomial Naive Bayes algorithm and TF-IDF vectorization. The model achieved an accuracy of 96.23%, demonstrating its effectiveness in identifying spam messages from legitimate messages.
